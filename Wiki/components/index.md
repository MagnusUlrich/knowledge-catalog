# Components

The three main components in the knowledge-catalog repository.

# Components

* [okf-agent.md](okf-agent.md) - Standalone OKF enrichment agent — produces OKF bundles from BigQuery using Gemini (okf/ directory)
* [enrichment-agent.md](enrichment-agent.md) - Full Dataplex enrichment agent — production-grade metadata enrichment using Google ADK + Vertex AI (agents/enrichment/)
* [kcmd.md](kcmd.md) - kcmd Metadata as Code CLI — bidirectional sync between local YAML/Markdown and Google Dataplex Knowledge Catalog (agents/mdcode/)
